# 🤖 AI Text Classifier

A Machine Learning powered Text Classification Web App built using Flask, Scikit-Learn and TF-IDF Vectorization.

## 🚀 Features

- Classifies text into categories
- Flask Web Interface
- TF-IDF Text Vectorization
- Naive Bayes Machine Learning Model
- Responsive UI
- Real-time Prediction

## 🛠 Tech Stack

- Python
- Flask
- Scikit-Learn
- TF-IDF Vectorizer
- Multinomial Naive Bayes
- HTML
- CSS

## ▶️ Run Locally

Clone the project

```bash
git clone https://github.com/Gauravvarya/Ai-Text-Classifier.git
```

Go to project directory

```bash
cd Ai-Text-Classifier
```

Install dependencies

```bash
pip install flask scikit-learn
```

Run application

```bash
python app.py
```

Open browser

```text
http://127.0.0.1:5001
```
